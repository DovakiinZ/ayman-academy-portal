# Ayman Academy Portal

مؤسسة تعليمية أكاديمية متخصصة في تقديم تعليم متميز للمراحل التأسيسية.

An academic educational institution specialized in providing premium education for foundational stages.

## Technology Stack

- **Framework**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI + shadcn/ui
- **State Management**: TanStack Query

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
│   ├── home/       # Homepage sections
│   ├── layout/     # Header, Footer, Layout
│   └── ui/         # Base UI components
├── contexts/       # React contexts (Language)
├── hooks/          # Custom React hooks
├── pages/          # Page components
└── lib/            # Utilities
```

## Features

- **Bilingual Support**: Arabic (RTL) and English (LTR)
- **Educational Stages**: Kindergarten, Primary, Middle School
- **Video Lessons**: Preview and subscriber content
- **Instructor Profiles**: Faculty information
- **Subscription Plans**: Payment-ready structure
